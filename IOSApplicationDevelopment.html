IOS 10, WatchOS 3 & TVOS 10 Application Development (Swift 3.0).

  1. Home
  2. IT Courses

Apple Technologies Application Development

This course is for students who are

Innovative & passionate about building great apps, aims in improving quality of human life.

In this course you will be learning Swift 3 programming language , a powerful & intuitive programming language for macOs, iOs, watchOS & tvOs that lets everyone build amazing apps.

This course takes you through all the stages of building apps from Discover, Design, Develop, Distribute (testing & releasing in Apple's store) .

If you are an existing developer who wants to learn programming apple technologies or a fresher with no prior experience in programming this course is designed for you. This course is designed based on a philosophy learn by doing / experiencing , so that each section of this course is a series of self-paced practical's helps learners learn with experience.

# Syllabus
1. Introduction to iOS 10, Xcode 8 and Swift 3.0.
2. Getting Started with MVC; iOS, Xcode and Swift - Setting up environment.
3. Swift 3.0 in Detail Deep Dive (Using Playgrounds).
4. Project / Product Planning.
  1. Prototype design before code.
5. Advanced iOS Features - I.
  1. Table View ;Text Field ;ScrollView 
  2. Multi touch Gesture Handling 
  3. Combining MVCs; View Controller Lifecycle 
  4. Closures; Extensions; Protocols; Delegation
    5.  
  5. Multithreading; Timers
6. Advanced iOS Features - II.
  1. Core Data; Autolayout 
  2. Animation; Core Motion
  3. Core Location; Maps 
  4. More Segues; Visual Effects 
  5. Camera; Persistence 
7. Advanced iOS Features - III.
  1. Integrating iAds.
8. Advanced iOS Features - IV.
  1. Integrating In App Purchases.
10. Advanced iOS Features - IV.
  1. 3D Touch: Peek & Pop; Implementing Touch ID.
11. Project Work.
12. Submitting App To The App Store.
